                    ТРЕБОВАНИЯ К СБОРКЕ ПРОЕКТА
_____________________________________________________________________
Система:
    -OS Linux kernel version=3.16.0
Необходимо установить пакеты:
    -gcc 	version=4.9.3
    -g++ 	version=4.9.3 ( поддержка последнего стандарта с++14 )
    -cmake 	version=2.8.12.2
______________________________________________________________________

                     ОПИСАНИЯ СТРУКТУРЫ ПРОЕКТА
______________________________________________________________________
Общие папки:
~/clion_project_template/                       |-папка проекта
~/--------/build/                    		|-папка для сборки подпроекта
~/--------/build/bin/                           |-папка для исполняемых 
                                                  файлов
~/--------/build/CMakeLists.txt      		|-файл для утилиты cmake
~/--------/src/                      		|-папка для файлов реализации
                                                  подпроекта   ( *.cpp )
~/--------/inc/                      		|-папка для заголовочных файлы 
                                                  подпроекта ( *.h )
~/--------/Makefile                   		|-Makefile для сборки подпроекта
